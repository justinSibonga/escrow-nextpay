Escrow API Flow

Step-by-step interaction guide for handling escrow transactions from creation to release or refund.

1. Create Escrow

Creates a new escrow record and prepares the system to accept funds.

Endpoint

POST /escrows

Request Body

{ "amount_cents": 150000, "currency": "PHP", "payer_id": "user_001", "payee_id": "user_002", "metadata": { "order_id": "ORD-1001", "service": "Website Development" } }

Response

{ "escrow_id": "esc_123456789", "status": "created" }

2. Create Payment Intent

Creates a payment intent that collects funds from the payer and routes them into the escrow account.

Endpoint

POST /payment-intents

Request Body

{ "account_id": "escrow_account_id", "amount_cents": 150000, "currency": "PHP", "reference_number": "ESCROW-ORD-1001", "metadata": { "escrow_id": "esc_123456789" } }

Result

  • Customer completes payment
  • Funds are credited to the escrow account
  • Escrow Status: funded

3. Payment Confirmation

The system listens for payment confirmation before locking the funds.

Webhook Event

payment_intent.succeeded

Action

  • Verify escrow_id from metadata
  • Lock funds in escrow
  • Prevent withdrawal by payee
  • Escrow Status: holding

4. Release Escrow

Once conditions are met, release the escrow funds to the payee.

Endpoint

POST /escrows/{escrow_id}/release

Action

  • Create a payout request
  • Transfer funds from escrow to payee account
  • Escrow Status: released

5. Payout Processing

Track payout progress via webhooks.

Possible Events

payout.processed payout.succeeded payout.failed

Final State

  • On success → funds received by payee
  • On failure → escrow enters manual review

6. Refund Escrow (Optional)

Refunds the payer if the transaction is canceled or conditions fail.

Endpoint

POST /escrows/{escrow_id}/refund

Action

  • Funds are returned to the payer
  • Escrow is closed
  • Escrow Status: refunded

Error Handling

Scenario Result
Payment failed Escrow remains created
Payment expired Escrow canceled
Payout failed Escrow flagged for review
Duplicate release Idempotent response
